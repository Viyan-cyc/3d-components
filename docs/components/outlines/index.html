Outlines

extends THREE.Group implements IDisposable

描边（轮廓）组件。取父级 Mesh 的几何体，按折痕角度分裂法线后生成一个背面渲染的描边网格，在顶点着色器里沿法线把几何外扩 thickness，再叠回父级之下，形成轮廓描边效果。算法移植自 pmndrs/drei 的 Outlines，但实现为原生 Three.js 类，不依赖 React/R3F。支持普通 Mesh、SkinnedMesh（绑定骨架）、InstancedMesh（复用实例矩阵）。

Constructor

new Outlines(options: OutlinesOptions)

关于 screenspace：裁剪空间模式（默认）下，描边厚度会随摄像机远近变化——近处更粗、远处更细，体积感更自然；屏幕空间模式下线宽恒定，适合 UI / 卡通描边风格。后者需要调用 attachRenderer() 让组件读取绘图缓冲尺寸做归一化。

关于 angle：Three.js 默认法线是平滑的（顶点法线 = 邻接面法线平均），直接拿来做描边会让相邻面的描边互相「吃掉」，平面区域也被描边。toCreasedNormals 在折痕角度 > angle 的硬边处把顶点分裂成多份，使每个面拥有独立法线，描边只沿真实轮廓外扩。angle = 0 则跳过此步、直接复用父级几何（不拷贝、不释放父级资源）。

Methods

.attachRenderer(renderer: THREE.WebGLRenderer): void

绑定渲染器以读取绘图缓冲尺寸（screenspace 偏移归一化用），并监听 canvas resize。仅在 screenspace = true 时有必要调用；不传则 size 默认为 1×1。

.setAngle(angle: number): void

变更折痕角度并重建描边网格（仅当与当前值不同时才重建）。0 = 不分裂法线。

.refresh(): void

父级几何体被替换后，调用此方法重建描边（旧的自建几何体会被释放）。

.dispose(): void

释放描边几何体与材质，并断开 resize 监听。不会触碰父级网格的资源（几何体 / 材质由父级自行管理）。

示例

import { Outlines } from '@cyc/3d-components/core';

// 给一个普通网格加描边
const mesh = new THREE.Mesh(geometry, material);
const outline = new Outlines({ mesh, color: 'red', thickness: 0.05 });
mesh.add(outline); // 描边作为 mesh 的子级，自动跟随变换 / 骨架
scene.add(mesh);

// 屏幕空间恒定线宽（卡通描边风格）
const toon = new Outlines({
  mesh, color: 'black', thickness: 0.01, screenspace: true,
});
toon.attachRenderer(renderer); // 让组件读取画布尺寸做归一化
mesh.add(toon);

// 切换折痕角度（运行时重建）
outline.setAngle(Math.PI / 4);
