Shape

extends THREE.Group implements IDisposable

异形面绘制组件。接收 XZ 平面上的轮廓点，挤出指定高度，生成带圆角拐角的实心几何体。轮廓总是闭合（首尾自动相连）。适合台面、地板异形区域、建筑平台等场景。

Constructor

new Shape(options: ShapeOptions)

Parameters — ShapeOptions

参数类型默认值说明
shapesShapeData[]—必填。一组异形面数据，每个元素生成一个挤出几何体
materialTHREE.Material—共享材质（所有异形面复用）。不传则使用默认 MeshStandardMaterial
namestring—Object3D.name
visiblebooleantrue是否可见
userDataobject—自定义数据
childrenObject3D[]—额外子节点，自动 add()

Data — ShapeData（单个异形面）

字段类型默认值说明
path[number, number, number][]—XZ 平面轮廓点 [x, y, z]（y 被忽略），首尾自动闭合。至少 3 个点
heightnumber—挤出高度（沿 Y 轴）
radiusnumber | (number | undefined)[]0拐角圆角半径。数值 ⇒ 所有拐角统一；数组 ⇒ 按 path 顶点索引逐拐角指定，undefined 回退到全局值
radiusSegmentsnumber8圆角分段数（越大越圆滑）
uvMode'repeat' | 'stretch''repeat'UV 映射方式：'repeat' = 按米平铺，'stretch' = 一张贴图铺满

圆角半径规则

radius 支持两种形式：

  • 标量（如 0.3）：所有拐角使用同一半径
  • 数组（如 [0.5, undefined, 1, undefined]）：按 path 顶点索引逐拐角指定。undefined 元素回退到标量值（即全局 radius = 0）。有单独设置用单独的，没有就用全局值。

UV 映射

异形面包含侧面和顶/底面，UV 分别处理：

  • 侧面：u = 沿轮廓弧长、v = 高度。与 Wall / Path 的 uvMode 一致：'repeat' 按米平铺，'stretch' 归一化 [0,1]
  • 顶/底面：平面投影 UV，u = x 范围、v = z 范围。'repeat' 按米平铺，'stretch' 归一化 [0,1]

Methods

.dispose(): void

释放所有异形面几何体；若材质由本组件创建则一并释放（外部传入的 material 不会被释放）。

示例

import { Shape } from '@cyc/3d-components/core';

// L 形异形台面，统一圆角
const shape = new Shape({
  shapes: [{
    path: [[0,0,0], [4,0,0], [4,0,2], [2,0,2], [2,0,5], [0,0,5]],
    height: 0.6,
    radius: 0.3,
    radiusSegments: 12,
    uvMode: 'repeat',
  }],
});
scene.add(shape);

// 混合圆角：第 0、3 个拐角 0.5，其余用全局值 0
const shape2 = new Shape({
  shapes: [{
    path: [[0,0,0], [3,0,0], [3,0,3], [0,0,3]],
    height: 0.4,
    radius: [0.5, undefined, undefined, 0.5],
    radiusSegments: 12,
  }],
});
scene.add(shape2);
