BaseGroup

extends THREE.Group implements IUpdatable, IDisposable

通用 Group 包装器。接受声明式配置，自动处理子节点添加、缩放、资源释放。适合作为所有 group 类组件的基础。

Constructor

new BaseGroup(options?: BaseGroupOptions)

Parameters — BaseGroupOptions

参数类型默认值说明
namestring—Object3D.name
scalenumber1初始缩放 (uniform)
visiblebooleantrue是否可见
userDataobject—自定义数据
childrenObject3D[]—子节点数组，自动 add()

Methods

.update(delta: number): void

每帧调用。在子类中覆盖实现逐帧行为。默认空操作。

参数类型说明
deltanumber距上一帧秒数（上限 100ms）
.dispose(): void

释放所有资源。遍历子节点释放 geometry / material，清空 children。覆盖时最后调用 super.dispose()。
