Sky

extends THREE.Mesh implements IDisposable

天空穹顶。基于内翻球体（BackSide）+ 着色器实现，通过从地平线到天顶的幂次渐变模拟天空大气效果。球体始终从内部可见，相机应位于球体内部。天顶与地平线颜色可独立配置，渐变指数与偏移量可灵活调整渐变曲线。

Constructor

new Sky(options?: SkyOptions)

Parameters — SkyOptions

参数类型默认值说明
sizenumber1000天空球半径（世界单位），应足够大以包围场景但不超过相机远裁剪面
topColorColorRepresentation0x109df4天顶颜色（球体最高处的颜色），天蓝
bottomColorColorRepresentation0xf5f5f5地平线 / 底部颜色，浅灰白
offsetnumber0世界坐标偏移量，增大使渐变整体上移，地平线附近更偏 bottomColor
exponentnumber0.6渐变指数，值越大过渡越陡峭（天顶色区域更集中），值越小过渡越平缓
namestring—Object3D.name
visiblebooleantrue是否可见
userDataobject—自定义数据

Methods

.setColors(top: ColorRepresentation, bottom: ColorRepresentation): this

同时设置天顶与地平线颜色。链式调用。

.setTopColor(color: ColorRepresentation): this

设置天顶颜色（hex / CSS 字符串 / THREE.Color）。链式调用。

.setBottomColor(color: ColorRepresentation): this

设置地平线 / 底部颜色。链式调用。

.setOffset(value: number): this

设置世界坐标偏移量。增大该值使渐变整体上移。链式调用。

.setExponent(value: number): this

设置渐变指数。值越大天顶色区域越集中，值越小过渡越平缓。链式调用。

.dispose(): void

释放 geometry 与 material。

示例

import { Sky } from '@cyc/3d-components/core';

// 天空穹顶
const sky = new Sky({
  size: 500,
  topColor: 0x109df4,
  bottomColor: 0xf5f5f5,
  offset: 0,
  exponent: 0.6,
});
scene.add(sky);

// 运行时调整颜色（所有 setter 链式调用）
sky.setTopColor(0x1a8af4).setBottomColor(0xe0e0e0);
sky.setOffset(10).setExponent(0.8);

// 同时设置天顶与地平线颜色
sky.setColors(0x109df4, 0xf5f5f5);
