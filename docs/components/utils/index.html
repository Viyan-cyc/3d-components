Util

Math / Color / Geometry 三大类工具函数。通过 Util.xxx() 调用，也支持按名单独导入（tree-shaking）。

import { Util } from '@cyc/3d-components/utils';
Util.clamp(5, 0, 10);
Util.createSphere(2, 500);

Math

函数签名说明
clamp(value, min, max): number钳制到 [min, max]
lerp(a, b, t): number线性插值 (t=0→a, t=1→b)
mapRange(v, inMin, inMax, outMin, outMax): number范围映射
degToRad(degrees): number度 → 弧度
radToDeg(radians): number弧度 → 度
randomRange(min, max): number[min, max) 随机浮点数
randomInt(min, max): number[min, max] 随机整数
distance(a: Vec3, b: Vec3): number3D 欧几里得距离
smoothstep(edge0, edge1, x): numberHermite 平滑插值

Color

函数签名说明
hexToRgb(hex): ColorRGBAhex → RGB (0–1)
rgbToHex(r, g, b): stringRGB (0–255) → "#rrggbb"
hslToRgb(h, s, l): ColorRGBAHSL (0–1) → RGB (0–1)
blendColors(a, b, t): ColorRGBA混合颜色 (t=0→a, t=1→b)

Geometry

函数签名说明
createGrid(rows, cols, spacing): Vec3[]2D 网格点阵
createCircle(radius, segments): Vec3[]圆环点阵
createSphere(radius, count): Vec3[]Fibonacci 球面分布
createSpiral(turns, ptsPerTurn, radius): Vec3[]阿基米德螺线
