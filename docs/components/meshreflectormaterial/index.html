MeshReflectorMaterial

extends THREE.MeshStandardMaterial implements IDisposable

实时平面反射材质。基于 MeshStandardMaterial，通过虚拟镜像相机将场景渲染到 RenderTarget，再合成到表面。支持多通道模糊、深度感知模糊、法线贴图扰动和扭曲贴图。适用于镜面、地板反射、水面等效果。

Constructor

new MeshReflectorMaterial(options?: MeshReflectorMaterialOptions)

Parameters — MeshReflectorMaterialOptions

参数类型默认值说明
resolutionnumber256反射 RenderTarget 分辨率，越高越清晰但更耗性能
blurnumber | [number, number][0, 0]模糊半径，单值或 [宽, 高] 元组，0 为禁用
mixBlurnumber0模糊混合度 (0=仅清晰, 1=全模糊)
mixStrengthnumber1反射强度乘数
mixContrastnumber1反射对比度 (1=不变, >1=更高, <1=更低)
mirrornumber0镜面程度 (0=与基础色混合, 1=纯反射)
distortionnumber1扭曲强度，需配合 distortionMap 使用
distortionMapTHREE.Texturenull扭曲贴图，红通道作为 UV 偏移
minDepthThresholdnumber0.9深度感知模糊最小阈值
maxDepthThresholdnumber1深度感知模糊最大阈值
depthScalenumber0深度缩放 (>0 启用深度感知模糊)
depthToBlurRatioBiasnumber0.25深度与模糊比率偏置
reflectorOffsetnumber0反射面沿法线偏移，避免 z-fighting
colorColorRepresentation0xffffff基础颜色
metalnessnumber0金属度 0–1
roughnessnumber0粗糙度 0–1 (也影响模糊混合)

核心方法

.updateBeforeRender(renderer: WebGLRenderer, scene: Scene, camera: PerspectiveCamera): boolean

每帧在 renderer.render() 之前调用，更新反射纹理。渲染场景到虚拟镜像相机，应用斜裁剪平面，可选模糊。返回 true 表示反射已更新，false 表示反射面背对相机（无需更新）。

参数类型说明
rendererWebGLRendererWebGL 渲染器
sceneScene要反射的场景
cameraPerspectiveCamera主相机（用于计算虚拟相机位置）
.bindToMesh(mesh: THREE.Mesh): this

将材质绑定到使用它的 Mesh 对象。必须在将材质赋给 Mesh 后调用，以便材质知道反射平面的位置。支持链式调用。

链式设置方法

.setMirror(v: number): this

设置镜面程度 (0=与基础色混合, 1=纯反射)。

.setMixBlur(v: number): this

设置模糊混合度 (0=仅清晰, 1=全模糊)。

.setMixStrength(v: number): this

设置反射强度乘数。

.setMixContrast(v: number): this

设置反射对比度。

.setDistortion(v: number): this

设置扭曲强度。

资源释放

.dispose(): void

释放所有 GPU 资源（RenderTarget、模糊通道等）。调用后材质不可再使用。

代码示例

import * as THREE from 'three';
import { MeshReflectorMaterial } from '@cyc/3d-components/material';

// 创建反射地板
const reflectorMat = new MeshReflectorMaterial({
  mirror: 0.75,
  blur: [300, 100],
  mixBlur: 1,
  mixStrength: 0.8,
  resolution: 512,
  color: 0x999999,
});
const floor = new THREE.Mesh(
  new THREE.PlaneGeometry(10, 10),
  reflectorMat,
);
floor.rotation.x = -Math.PI / 2;
reflectorMat.bindToMesh(floor);
scene.add(floor);

// 渲染循环中更新反射
function animate() {
  requestAnimationFrame(animate);
  reflectorMat.updateBeforeRender(renderer, scene, camera);
  renderer.render(scene, camera);
}
