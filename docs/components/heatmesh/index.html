HeatMesh

extends THREE.Mesh implements IUpdatable, IDisposable

热力图可视化 Mesh。以半透明球体表示热力区域，支持动态强度控制、GSAP 动画、每帧自动脉冲效果。

Constructor

new HeatMesh(options?: HeatMeshOptions)

Parameters — HeatMeshOptions

参数类型默认值说明
namestring—Object3D.name
countnumber64球体顶点分辨率
radiusnumber1热力区域半径
intensitynumber0.5初始强度 0–1
visiblebooleantrue是否可见
userDataobject—自定义数据

Properties

属性类型说明
heatIntensitynumber当前热力强度 0–1
heatRadiusnumber当前热力半径

Methods

.update(delta: number): void

每帧更新，应用基于 heatIntensity 的脉冲透明度效果。

.setIntensity(value: number): this

设置热力强度。自动钳制到 0–1。返回 this 支持链式调用。

参数类型说明
valuenumber新强度值，越界自动钳制
.dispose(): void

释放 geometry 和 material。
