InstancedMesh2

extends THREE.Mesh

高性能实例化渲染器。基于间接实例化（Indirect Instancing）架构，通过 instanceIndex 属性索引 DataTexture 中的逐实例矩阵、颜色、骨骼和自定义 Uniform，实现零缓冲区重分配的逐实例视锥裁剪、LOD 切换与部分 GPU 更新。内置 BVH 空间索引加速视锥裁剪、射线检测与 LOD 距离判定；支持逐实例骨骼动画、变形目标与自定义 Uniform。

Constructor

new InstancedMesh2<TData, TGeometry, TMaterial>(geometry, material, params?, LOD?)

Parameters — InstancedMesh2Params

参数类型默认值说明
capacitynumber1000缓冲区最大容量（实例数），超出时自动扩容
createEntitiesbooleanfalse是否创建 instances 实体数组，开启后可通过 InstancedEntity 逐实例操作 position / rotation / scale
allowsEulerbooleanfalse允许 InstancedEntity.rotation（Euler），与 quaternion 双向同步
rendererWebGLRenderer—若提供则立即初始化缓冲区；否则延迟到首次渲染

Methods — 实例管理

.addInstances(count: number, onCreation?: UpdateEntityCallback): this

添加实例。可选回调初始化每个新实例的位置/颜色等。返回 this 支持链式调用。

.removeInstances(...ids: number[]): this

按 ID 移除实例。ID 被回收复用。

.clearInstances(): this

清除所有实例。

.updateInstances(onUpdate: UpdateEntityCallback): this

遍历所有活跃实例，调用回调后更新矩阵（position + quaternion + scale）。

.updateInstancesPosition(onUpdate: UpdateEntityCallback): this

同上，但仅更新位置（更快，适合纯位移场景）。

.resizeBuffers(capacity: number): this

手动调整缓冲区容量。

Methods — 矩阵

设置指定实例的变换矩阵。

.getMatrixAt(id: number, matrix?: Matrix4): Matrix4

获取指定实例的变换矩阵。

.getPositionAt(index: number, target?: Vector3): Vector3

从矩阵中提取实例位置。

Methods — BVH

.computeBVH(config?: BVHParams): void

创建并计算 BVH 空间索引，加速视锥裁剪、射线检测和 LOD 距离判定。

.disposeBVH(): void

销毁 BVH。

BVHParams

参数类型默认值说明
marginnumber0包围盒外扩边距，适合动画/移动对象
getBBoxFromBSpherebooleanfalse从包围球计算 AABB（更快但精度低，仅适用于中心在原点的几何体）

Methods — LOD

.setFirstLODDistance(distance: number): this

初始化 LOD 系统，设置 LOD0（基础网格）的切换距离。必须在 addLOD 之前调用。

.addLOD(geometry: BufferGeometry, material: Material, distance?: number, hysteresis?: number): this

添加渲染 LOD 级别。每个级别创建一个共享数据纹理的子 InstancedMesh2。distance 为距离阈值（内部取平方），hysteresis 为滞后因子（0-1~1，防止 LOD 抖动）。

.addShadowLOD(geometry: BufferGeometry, distance?: number, hysteresis?: number): this

添加阴影专用 LOD 级别（castShadow = true）。

.updateLOD(levelIndex: number, distance?: number, hysteresis?: number): this

更新指定渲染 LOD 级别的参数（不可修改 LOD0）。

.updateAllLOD(distances?: number[], hysteresis?: number): this

批量更新所有渲染 LOD 距离/滞后。

.removeLOD(levelIndex: number, removeObject?: boolean): this

移除 LOD 级别（LOD0 存在时不可移除）。

Methods — 骨骼动画

.initSkeleton(skeleton: Skeleton, disableMatrixAutoUpdate?: boolean): void

初始化逐实例骨骼动画。创建 boneTexture，默认禁用骨骼的 matrixAutoUpdate 以提升性能。

.setBonesAt(id: number, updateBonesMatrices?: boolean, excludeBonesSet?: Set<string>): void

设置指定实例的骨骼矩阵。可排除特定骨骼不更新。

.setUniformAt(id: number, name: string, value: UniformValue): void

设置指定实例的 Uniform 值。

.getUniformAt(id: number, name: string, target?: UniformValueObj): UniformValue

获取指定实例的 Uniform 值。

Methods — 变形目标

InstancedEntity

当 createEntities: true 时，每个实例对应一个 InstancedEntity 对象，提供类似 Object3D 的操作接口：

属性类型说明
idnumber实例唯一 ID（只读）
positionVector3局部位置
scaleVector3局部缩放（默认 1,1,1）
quaternionQuaternion局部旋转（四元数）
rotationEuler | undefined局部旋转（Euler，需 allowsEuler: true）
visibleboolean可见性（代理到 owner）
activeboolean激活状态（代理到 owner）
colorColor颜色（代理到 owner）
opacitynumber透明度（代理到 owner）

示例

import { InstancedMesh2 } from '@cyc/3d-components/core';
import * as THREE from 'three';

// 基础用法：创建 5000 个实例
const mesh = new InstancedMesh2(
  new THREE.BoxGeometry(1, 1, 1),
  new THREE.MeshStandardMaterial({ color: 0x44aa88 }),
  { capacity: 5000, createEntities: true, allowsEuler: true },
);

mesh.addInstances(5000, (obj, i) => {
  obj.position.set(Math.random() * 100 - 50, 0, Math.random() * 100 - 50);
  obj.rotation.y = Math.random() * Math.PI * 2;
  obj.updateMatrix();
});
scene.add(mesh);

// 启用 BVH 加速视锥裁剪
mesh.computeBVH({ getBBoxFromBSphere: true });

// LOD：4 级细节，距离越远越简化
mesh.setFirstLODDistance(0);
mesh.addLOD(geoL1, matL1, 15);   // 15 单位内用 L1
mesh.addLOD(geoL2, matL2, 40);   // 40 单位内用 L2
mesh.addLOD(geoL3, matL3, 80);   // 80 单位内用 L3

// 逐实例颜色
mesh.setColorAt(0, new THREE.Color('red'));

// 逐实例自定义 Uniform
mesh.initUniformsPerInstance({
  vertex: { windStrength: 'float', windDir: 'vec2' },
});
mesh.setUniformAt(0, 'windStrength', 0.8);
mesh.setUniformAt(0, 'windDir', new THREE.Vector2(1, 0));
