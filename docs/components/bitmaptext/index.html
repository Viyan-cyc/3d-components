BitmapText

extends THREE.Mesh implements IDisposable

SDF 位图文字组件。基于 Canvas 2D 在运行时栅格化字符，经 DistanceTransform 生成 Signed Distance Field 图集，用自定义 GLSL 着色器渲染——任意缩放不锯齿，自带阴影、描边、CJK 自动换行。移植自 t3d.js BitmapTextMaterial + DynamicFont，无需字体文件或第三方插件。

开关投影。切换 USE_SHADOW 宏，首次启用需重新编译着色器（一次性开销）。链式调用。

原理说明

SDF 图集生成：组件内部使用 DynamicFont 在 Canvas 2D 上逐字符调用 fillText 栅格化字形，再经 DistanceTransform（Felzenszwalb & Huttenlocher 线性时间 EDT）将 alpha 通道转换为有符号距离场。所有字符打包进一张 DataTexture(RedFormat) 单通道纹理。

排版引擎：TextLayout 负责 word-wrap（英文按词断行 / CJK 逐字断行）、对齐（左 / 中 / 右）、字符间距和行高计算，输出每个字符的屏幕坐标和 UV。

着色器渲染：片段着色器对 SDF 距离值做 smoothstep(halo - γ, halo + γ, dist) 生成 alpha，先合成阴影层，再合成描边层，最后合成文字颜色。gamma 乘以 1.5 × length(fwidth(v_Size)) 实现屏幕空间自适应抗锯齿。

示例

import { BitmapText } from '@cyc/3d-components/core';

// 基础中文文本
const text = new BitmapText({
  text: 'Hello 世界',
  fontSize: 72,
  width: 1000,
  align: 'center',
});
scene.add(text);

// 带描边和阴影
const fancy = new BitmapText({
  text: '道可道，非常道',
  fontSize: 64,
  outline: true,
  outlineColor: 0x00bbff,
  outlineWidth: 0.06,
  shadow: true,
  scale: 0.01,
});
scene.add(fancy);

// 运行时更新
text.setText('新的文字内容');
text.setColor(0xffcc00);
fancy.setOutline(true);
fancy.setOutlineParams(0xff0000, 0.08, 1);

// 清理
text.dispose();
