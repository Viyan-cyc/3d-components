Html

extends THREE.Group implements IUpdatable, IDisposable

3D 空间中的 HTML 覆盖层组件。将 DOM 元素定位在 Three.js 场景的 3D 坐标上，每帧根据相机投影自动更新屏幕位置。支持两种定位模式：2D 投影（默认）和 CSS3D 变换。参考 drei Html 组件，以纯 Three.js 重写，无 React 依赖。

定位模式说明

2D 投影模式（transform: false，默认）：通过 Vector3.project(camera) 将 3D 世界坐标投影为 NDC，再转换为屏幕像素坐标，用 CSS translate3d 定位 DOM 元素。简单高效，适合标签、信息卡片等 2D 覆盖场景。配合 distanceFactor 可实现近大远小的透视缩放。

CSS3D 变换模式（transform: true）：从 Three.js 相机矩阵和对象矩阵构建完整 matrix3d CSS 变换，实现 DOM 元素在 3D 空间中的真正透视变换——元素会随相机旋转产生倾斜、近大远小等 3D 效果，类似 CSS3DRenderer 但无需额外渲染器。外层 div 承载相机变换，内层 div 承载对象变换。

遮挡模式说明

occlude: true — 在 3D 场景中渲染一个不可见的平面网格（片段着色器输出全透明，但写入深度缓冲），3D 物体在 HTML 元素前方时会遮挡住 HTML。该网格会自动根据 HTML 元素尺寸调整大小。

occlude: 'raycast' — 每帧从相机向 HTML 元素发射射线，检测是否被场景中的几何体遮挡。若被遮挡则隐藏 HTML 元素（或触发 onOcclude 回调）。

occlude: 'blending' — 通过调整 canvas 的 z-index 将其置于 HTML 层之间，使 3D 场景与 HTML 产生混合合成效果。

occlude: [obj1, obj2, ...] — 对指定 Object3D 数组做射线遮挡检测，而非整个场景，性能更优。

示例

import { Html } from '@cyc/3d-components/core';

// ── 基础标签 ──
const label = new Html({ center: true });
label.position.set(0, 2, 0);
label.element.innerHTML = '<div style="background:#fff;padding:8px 16px;border-radius:4px;box-shadow:0 2px 8px rgba(0,0,0,.15)">Hello 3D!</div>';
scene.add(label);

// ── 带距离缩放 ──
const info = new Html({
  distanceFactor: 10,
  center: true,
  style: 'background:rgba(0,0,0,.75);color:#fff;padding:10px;border-radius:6px;',
});
info.position.set(1, 1.5, -1);
info.element.innerHTML = '距离缩放标签';
scene.add(info);

// ── CSS3D 变换模式 ──
const panel = new Html({
  transform: true,
  distanceFactor: 10,
});
panel.position.set(0, 1.2, -3);
panel.element.innerHTML = '<div style="width:300px;background:white;padding:20px">3D 面板</div>';
scene.add(panel);

// ── 射线遮挡 ──
const occluded = new Html({
  center: true,
  occlude: 'raycast',
  onOcclude: (hidden) => console.log('遮挡状态:', hidden),
});
occluded.position.set(0, 0.5, 0);
scene.add(occluded);

// ── 渲染循环 ──
function animate() {
  requestAnimationFrame(animate);
  const delta = clock.getDelta();
  label.update(delta, camera, renderer);
  info.update(delta, camera, renderer);
  panel.update(delta, camera, renderer);
  occluded.update(delta, camera, renderer);
  renderer.render(scene, camera);
}

// ── 清理 ──
label.dispose();
info.dispose();
