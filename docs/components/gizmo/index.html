GizmoHelper

implements IUpdatable implements IDisposable

视口导航 Gizmo（Viewport Gizmo）。参考 drei GizmoHelper 的原生 Three.js 实现：在屏幕角落绘制一个独立小视口，用正交相机渲染一个三轴指示器（GizmoViewport），实时镜像主相机朝向；点击轴头即可把主相机平滑旋转到对应标准视角。GizmoHelper 自带虚拟场景与射线拾取，不要 scene.add(gizmo)。

Constructor

new GizmoHelper(options: GizmoHelperOptions)

Parameters — GizmoHelperOptions

参数类型默认值说明

contentGizmoContent—gizmo 内容（如 GizmoViewport），也可构造后用 setContent 设置
alignmentGizmoAlignment'bottom-right'九宫格对齐位置
margin[number, number][16, 16]相对屏幕边缘的留白 [水平, 垂直]（CSS 像素）
sizenumber120gizmo 叠层方形边长（CSS 像素）
disabledbooleanfalse是否整体禁用（不渲染、不响应点击、不做同步）
onTarget() => Vector3—自定义轨道目标点提供函数，优先级高于 controls.target
onUpdate() => void—每个动画步的回调，替代 controls.update

Methods

.setContent(content: GizmoContent): this

设置 gizmo 内容（替换现有内容）。链式调用。

.update(delta: number): void

每帧调用：把 gizmo 朝向同步到主相机、调用内容的逐帧回调，并推进相机动画（若有）。纯变换逻辑，不绘制。

.renderOverlay(): void

把 gizmo 绘制到屏幕角落叠层。必须在每帧主渲染 renderer.render(scene, camera) 之后调用。

.tweenCamera(direction: Vector3): this

平滑动画主相机，使其从 direction 方向看向目标点。通常由内容的 onPick 调用。链式调用。

.setAlignment(a: GizmoAlignment): this

设置对齐位置。链式调用。

.setMargin(m: [number, number]): this

设置屏幕边缘留白。链式调用。

.setSize(s: number): this

设置叠层方形边长。链式调用。

.setDisabled(d: boolean): this

启用 / 禁用整个 gizmo。链式调用。

.dispose(): void

移除监听、释放内容与虚拟场景中的几何体 / 材质 / 贴图。

GizmoViewport

三轴视口指示器（GizmoHelper 的默认内容），仿 ThreeOrbitControlsGizmo 的 2D 扁平气泡样式。由 6 个彩色扁平圆形气泡（正轴亮、带暗色字母 X/Y/Z；负轴暗、较小）+ 从中心连出的细彩色线组成。每帧按各轴朝向切换正面=亮色 / 背面=暗色，鼠标悬停的气泡填白。气泡为始终面向相机的 Sprite。

new GizmoViewport(options?: GizmoViewportOptions)
参数类型默认值说明
onPick(dir: Vector3) => void—点击轴头回调，参数为该轴方向，通常 (dir) => gizmo.tweenCamera(dir)
colors{ x, y, z }见下三轴颜色，每条为 [正面亮色, 背面暗色]。默认 x #f73c3c/#942424、y #6ccb26/#417a17、z #178cf0/#0e5490
labels[string, string, string]['X','Y','Z']正轴气泡标签文字（常显），空串隐藏
negativeLabels[string, string, string]['-X','-Y','-Z']负轴气泡标签文字（仅悬停时显示）
labelColorColorRepresentation'#222222'标签文字常规色（正轴未悬停时）
hoverColorColorRepresentation'#ffffff'悬停时文字变白的颜色
negativeOpacitynumber0.35负轴气泡填充透明度
backdropOpacitynumber0.13整体悬停时白色圆底透明度（0 关闭）
sizenumber1气泡整体缩放

disabledbooleanfalse禁用点击拾取

示例

import { GizmoHelper, GizmoViewport } from '@cyc/3d-components/helper';

const gizmo = new GizmoHelper({ camera, renderer, controls });
gizmo.setContent(new GizmoViewport({
  onPick: (dir) => gizmo.tweenCamera(dir),
}));

// 渲染循环（renderOverlay 必须在主渲染之后）：
function frame() {
  const dt = clock.getDelta();
  controls.update();
  gizmo.update(dt);          // 同步 gizmo 朝向 + 动画相机
  renderer.render(scene, camera);
  gizmo.renderOverlay();     // 叠层绘制
  requestAnimationFrame(frame);
}
