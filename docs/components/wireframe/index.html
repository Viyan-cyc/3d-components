Wireframe

extends THREE.Group implements IDisposable

线框化组件。基于重心坐标（barycentric）在每个三角形内计算到三条边的最小距离，用 fwidth 抗锯齿画出线条；支持虚线、收窄、正反面不同色、半透明填充。算法移植自 pmndrs/drei 的 WireframeMaterial，实现为原生 Three.js 类，不依赖 React/R3F。

提供两种模式：独立线框（默认，overrideMaterial = false）在父级网格下挂一个子 Mesh，用独立 ShaderMaterial 渲染线框 + 填充，父级原材质不变；材质覆盖（overrideMaterial = true）通过 onBeforeCompile 把线框注入父级材质，与原 PBR 光照混合渲染，适合「带光照的线框模型」。

两种模式都要求几何体具备 barycentric 属性——本组件会在初始化时自动调用 applyBarycentric 计算（就地修改父级几何体：索引几何会被转为非索引，必要时请传入副本）。

Methods

.setUniform(key: keyof WireframeOptions, value: unknown): this

运行时更新某个 uniform（链式）。key 如 'thickness'、'stroke'、'dash'；颜色项接受 ColorRepresentation。

.dispose(): void

释放资源。独立模式：释放线框几何体副本与线框材质；覆盖模式：仅释放线框材质（uniform 容器），不还原父级材质 / 几何体（onBeforeCompile 注入无法安全撤销；如需还原请重建父级材质）。

示例

import { Wireframe } from '@cyc/3d-components/core';

// 独立线框（父级材质不受影响）
const mesh = new THREE.Mesh(geometry, standardMaterial);
const wf = new Wireframe({
  mesh, stroke: '#ff0000', thickness: 0.1, dash: true,
});
mesh.add(wf); // 作为子级，自动跟随父级变换
scene.add(mesh);

// 带光照的线框模型（注入 PBR 材质）
const lit = new Wireframe({
  mesh, overrideMaterial: true,
  fillMix: 0.5,    // 一半原色 / 一半 fill
  stroke: '#111111', fill: '#88ccff',
});
mesh.add(lit);

// 运行时调参（链式）
wf.setUniform('thickness', 0.2).setUniform('stroke', 'blue');
