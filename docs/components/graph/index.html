Graph3D

extends BaseGroup（THREE.Group） implements IUpdatable · IDisposable

基于图数据结构（节点 Node + 边 Edge）的 3D 可视化组件，设计理念参考 AntV G6，渲染引擎基于 Three.js。本页为第一步交付：核心数据类型（Node/Edge/Graph + 标准 3D 坐标接口 NodePos3D）、数据适配层（校验/归一化/邻接表）、Graph3D 骨架，渲染球体节点 + 直线段边。后续步骤将接入多形态元素、交互拾取、以及完整的 3D 布局算法体系。

上方 demo 默认渲染一组示例图（约 12 节点 / 18 边，初始在原点环形散布）。点「重新生成随机图」验证 setData 的重建与资源释放。

Methods

.setData(data: GraphData): void

校验并归一化数据，释放旧元素后重新生成球体节点与直线段边。初始坐标取 data.x/y/z，未指定则在原点环形散布。

.getNodes(): Node3D[]  ·  .getNode(id): Node3D | undefined

访问节点元素实例（便于后续步骤追加交互）。

.getEdges(): Edge3D[]  ·  .getData(): GraphData | null  ·  .getIndex(): GraphIndex | null

访问边实例与规范化数据/索引（邻接表）。

.update(delta: number): void  ·  .dispose(): void

update 每帧由渲染循环调用，转发给子元素；dispose 释放全部节点/边资源。

数据类型

interface GraphData {
  nodes: NodeData[];   // { id, x?, y?, z?, size?, type?, data? }
  edges: EdgeData[];   // { source, target, id?, type?, data? }
}

// 所有布局算法的统一输出 —— 含完整 x/y/z
interface NodePos3D { id: string | number; x: number; y: number; z: number; }

示例

import { Graph3D } from '@cyc/3d-components/graph';

const graph = new Graph3D({ nodeSize: 0.4 });
graph.setData({
  nodes: [{ id: 'n1' }, { id: 'n2' }, { id: 'n3' }],
  edges: [
    { source: 'n1', target: 'n2' },
    { source: 'n2', target: 'n3' },
  ],
});
scene.add(graph);

// 渲染循环中：
graph.update(delta);
