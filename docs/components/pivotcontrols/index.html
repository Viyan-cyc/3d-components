PivotControls

extends THREE.Group implements IUpdatable implements IDisposable

统一变换操控 Gizmo（平移 / 旋转 / 缩放一体）。参考 drei PivotControls 的原生 Three.js 实现：在一个 gizmo 上同时呈现三根轴箭头（单轴平移）、三个平面滑块（双轴平移）、三段旋转弧（绕轴旋转）和三个缩放球（单轴缩放）。把受控内容 pivot.add(model)，拖拽 gizmo 即变换该组；每帧调用 pivot.update(dt)。

Methods

.update(delta?: number): void

每帧调用：刷新世界矩阵、fixed 模式恒定像素缩放、FatLine 分辨率与注释定位。须在渲染前调用。

.recomputeAnchor(): void

重算 anchor 定位（动态 add 内容后调用一次）。

.setEnabled(e: boolean): this

启用 / 禁用交互（gizmo 仍显示）。链式。

.setVisible(v: boolean): this

显示 / 隐藏 gizmo。链式。

.dispose(): void

移除指针监听、释放操控件的几何 / 材质 / 注释（不释放你 add 的内容）。

操控件

四类操控件，覆盖完整变换。轴箭头与缩放球沿 +X/+Y/+Z；平面滑块与旋转弧作用于各坐标平面（法线轴着色）。

  • AxisArrow —— 拖一根轴箭头，沿该轴平移。
  • PlaneSlider —— 拖一个平面方框，在该平面内自由平移。
  • AxisRotator —— 拖一段弧，绕该面法线旋转（按住 Shift 吸附到 10°）。
  • ScalingSphere —— 拖一个球，沿该轴缩放（按住 Shift 吸附到 0.1）。

示例

import { PivotControls } from '@cyc/3d-components/controls';

const pivot = new PivotControls({ camera, renderer, controls: orbit });
scene.add(pivot);
pivot.add(model);

function frame() {
  const dt = clock.getDelta();
  orbit.update();
  pivot.update(dt);          // 每帧调用
  renderer.render(scene, camera);
  requestAnimationFrame(frame);
}
