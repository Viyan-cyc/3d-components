快速开始

安装

本库依赖 three 和 gsap（peer dependencies）:

npm install three gsap
npm install @cyc/3d-components

引入方式

支持整包引入和按需引入（tree-shaking），兼容 ESM / CJS：

整包 (ESM)

import { BaseGroup, HeatMesh, ShinyMaterial, Util } from '@cyc/3d-components';

按需引入（推荐）

import { BaseGroup } from '@cyc/3d-components/core';
import { HeatMesh } from '@cyc/3d-components/heat';
import { ShinyMaterial } from '@cyc/3d-components/material';
import { Util } from '@cyc/3d-components/utils';

CommonJS

const { BaseGroup } = require('@cyc/3d-components/core');
const { Util } = require('@cyc/3d-components/utils');

快速示例

import * as THREE from 'three';
import { BaseGroup } from '@cyc/3d-components/core';
import { ShinyMaterial } from '@cyc/3d-components/material';

const widget = new BaseGroup({ name: 'widget', scale: 1.5 });
widget.add(new THREE.Mesh(
  new THREE.SphereGeometry(0.5, 32, 32),
  new ShinyMaterial({ color: 0x3366ff, metalness: 0.3 })
));
scene.add(widget);

Wall

extends THREE.Group implements IDisposable

墙体绘制组件。接收一组墙体数据，为每面墙生成带圆角拐角、可选墙洞的挤出几何体。路径定义在 XZ 平面（y 被忽略），墙体从 y = 0 向上生长。适合建筑户型、园区围栏、室内隔墙等场景。

Constructor

new Wall(options: WallOptions)

Parameters — WallOptions

参数类型默认值说明
wallsWallData[]—必填。一组墙体数据，每个元素绘制一面墙
materialTHREE.Material—共享材质（所有墙体复用）。不传则使用默认 MeshStandardMaterial
namestring—Object3D.name
visiblebooleantrue是否可见
userDataobject—自定义数据
childrenObject3D[]—额外子节点，自动 add()

Data — WallData（单面墙）

字段类型默认值说明
path[number, number, number][]—墙体中心线路径，XZ 平面坐标 [x, y, z]（y 被忽略）。路径定义走向，墙体按 width 向两侧加厚
widthnumber—墙体厚度（向路径两侧各加厚 width / 2）
heightnumber—墙体高度（沿 Y 轴挤出）
radiusnumber | number[]0拐角圆角半径。数值 ⇒ 所有拐角统一；数组 ⇒ 按 path 顶点索引逐拐角单独指定（元素 0 = 直角）。小于 width/2 的半径自动按直角处理
radiusSegmentsnumber8圆角分段数（越大越圆滑）
closebooleanfalse首尾是否闭合（闭合时形成环形墙体）
holeWallHole[]—墙洞（窗 / 门）列表，画在墙的立面上，按 segment 归属到对应墙段并贯通墙体厚度

Data — WallHole（单个墙洞 · 窗 / 门）

墙洞画在墙的立面上（沿墙方向 × 高度），沿墙体厚度方向贯通。每个墙洞归属于某一段墙体，用该段的局部 2D 坐标描述轮廓。

字段类型默认值说明
segmentnumber—所属墙段索引（段 i = path[i] → path[i+1]）
path[number, number][]—轮廓，该段局部坐标 [x, y]：x = 沿直墙段距离（从段起点计），y = 高度（0 为墙底）。首尾自动闭合
radiusnumber0拐角圆角半径
radiusSegmentsnumber8圆角分段数

Methods

.dispose(): void

释放所有墙体几何体；若材质由本组件创建则一并释放（外部传入的 material 不会被释放）。

示例

import { Wall } from '@cyc/3d-components/core';

const wall = new Wall({
  walls: [
    {
      // 闭合矩形房间，带圆角
      path: [[0, 0, 0], [6, 0, 0], [6, 0, 5], [0, 0, 5]],
      width: 0.25,
      height: 3,
      radius: 1,
      radiusSegments: 16,
      close: true,
      hole: [
        // 第 0 段（[0,0,0]→[6,0,0]）上的一扇门：沿墙 2~4m、高 0~2.1m
        { segment: 0, path: [[2, 0], [4, 0], [4, 2.1], [2, 2.1]] },
        // 第 1 段（[6,0,0]→[6,0,5]）上的一扇窗：沿墙 1.5~3.5m、离地 1~1.8m
        { segment: 1, path: [[1.5, 1], [3.5, 1], [3.5, 1.8], [1.5, 1.8]], radius: 0.15 },
      ],
    },
    {
      // 开放隔墙，圆角拐角（首尾不闭合）
      path: [[-2, 0, 8], [2, 0, 8], [2, 0, 11]],
      width: 0.2,
      height: 2.5,
      radius: 0.8,
    },
  ],
});
scene.add(wall);

Grid

extends THREE.Mesh implements IDisposable

无限参考网格。基于屏幕空间四边形 + 着色器实现（移植自 t3d.js InfiniteGridShader），把整屏四边形反投影成世界射线、与网格平面求交来绘制无边界网格——无论相机如何移动或拉远，网格始终铺满屏幕。写入 gl_FragDepth，能被场景中的物体正确遮挡。自带主 / 次双层网格、坐标轴高亮（X 红、Z/Y 蓝）与距离淡出。

Constructor

new Grid(options?: GridOptions)

Parameters — GridOptions

参数类型默认值说明
primaryScalenumber5主网格（粗线）间距缩放，每隔该距离一条粗线
secondaryScalenumber1次级网格（细线）间距缩放，叠加在主网格上形成细分
showAxisbooleantrue是否显示坐标轴高亮（X 轴 + Z/Y 轴）。关闭后仅显示普通网格
primaryFadenumber0.7主网格淡出系数，乘到主网格线条 alpha 上
secondaryFadenumber0.4次级网格淡出系数，乘到次级网格线条 alpha 上
linearFadebooleantrue是否启用基于相机距离的线性淡出（远处网格逐渐消失）
fadeStartnumber30线性淡出起始距离（世界单位），小于该距离不淡出
fadeEndnumber100线性淡出结束距离（世界单位），大于该距离完全透明
plane'xz' | 'xy''xz'网格平面：'xz' 水平地面（Y=0），'xy' 面向 +Z 的立面（Z=0）
colorColorRepresentation0x333333网格线颜色（直接决定最终线色，颜色越深线越暗）
xAxisColorColorRepresentation0xff0000X 轴高亮颜色
zAxisColorColorRepresentation0x0000ffZ 轴（地面模式）/ Y 轴（立面模式）高亮颜色
namestring—Object3D.name
visiblebooleantrue是否可见
userDataobject—自定义数据

Properties

属性类型说明
plane'xz' | 'xy'当前网格平面（只读，用 setPlane() 修改）

Methods

.setPlane(plane: 'xz' | 'xy', animate?: boolean): this

切换网格平面。'xz' → 地面，'xy' → 立面。默认用 GSAP 平滑过渡（约 0.6s），传 animate = false 立即切换。

.setPrimaryScale(scale: number): this

设置主网格间距。链式调用。

.setSecondaryScale(scale: number): this

设置次级网格间距。链式调用。

.setShowAxis(show: boolean): this

开关坐标轴高亮。false 时仅显示普通网格。链式调用。

.setColor(color: ColorRepresentation): this

设置网格线颜色（hex / CSS 字符串 / THREE.Color）。链式调用。

.setXAxisColor(color: ColorRepresentation): this

设置 X 轴高亮颜色。链式调用。

.setZAxisColor(color: ColorRepresentation): this

设置 Z 轴（地面模式）/ Y 轴（立面模式）高亮颜色。链式调用。

.setFade(start: number, end: number): this

设置线性淡出起止距离。链式调用。

.setLinearFade(enabled: boolean): this

开启 / 关闭距离淡出。通过编译宏 USE_LINEARFADE 控制，切换时重新编译着色器（一次性开销）。链式调用。

.onBeforeRender(renderer, scene, camera): void

由 Three.js 渲染前自动调用，按当前相机刷新反投影矩阵。无需手动调用。

.dispose(): void

释放 geometry 与 material，并终止进行中的 GSAP 补间。

示例

import { Grid } from '@cyc/3d-components/core';

// 作为无限地面参考网，自定义网格与坐标轴颜色
const grid = new Grid({
  primaryScale: 5,
  secondaryScale: 1,
  color: '#9aa7b8',       // 网格线颜色
  xAxisColor: 0xff5555,   // X 轴
  zAxisColor: 0x55aaff,   // Z / Y 轴
});
scene.add(grid);

// 运行时调整样式（所有 setter 链式调用）
grid.setShowAxis(false).setLinearFade(true).setFade(20, 80);
grid.setColor('#ffffff').setXAxisColor('red');

// 平滑切换到 2D 立面网格（GSAP 动画）
grid.setPlane('xy');

Path

extends THREE.Group implements IDisposable

路径绘制组件。把一组 3D 顶点先经「直线段 + 二次贝塞尔圆角」构造曲线，再经 Frenet 标架采样，最后扫掠成管道（tube）或扁平带（plane）两种几何体。算法移植自 t3d.js 的 CurvePath3 / TubeBuilder / RouteBuilder。适合管线、电路走线、路网导航带、3D 路径动画轨道等场景。

上方 demo 的路径数据采用 2D Hilbert 曲线（移植自 t3d.js CurveUtils.hilbert2D），参考 t3d geometry_builder_lines 示例。

Constructor

new Path(options: PathOptions)

Parameters — PathOptions

参数类型默认值说明
pathsPathData[]—必填。一组路径数据，每个元素生成一条管道或扁平带
materialTHREE.Material—共享材质（所有路径复用）。不传则使用默认 MeshStandardMaterial（dispose() 时一并释放）
namestring—Object3D.name
visiblebooleantrue是否可见
userDataobject—自定义数据
childrenObject3D[]—额外子节点，自动 add()

Data — PathData（单条路径）

字段类型默认值说明
path[number, number, number][]—必填。路径顶点（3D）。相邻重复点自动去重；少于 2 个有效点时该条跳过
mode'tube' | 'plane''tube'生成模式：'tube' = 圆管扫掠；'plane' = 扁平带状面
bevelRadiusnumber0拐角圆角半径。> 0 时拐角用二次贝塞尔曲线倒圆角；0 时为纯折线（直角拐角）
closebooleanfalse路径是否自动闭合（首尾相连成环，需 ≥ 3 个顶点）
up[number, number, number]—up 向量，决定标架法向（进而决定平面带朝向与管道截面起始朝向）。不传则用 Frenet 自动选取（与 three.js TubeGeometry 一致）
divisionsnumber12非直线段（贝塞尔曲线）的采样分段数
uvMode'repeat' | 'stretch''repeat'贴图沿路径方向的映射：'repeat' = 按物理长度平铺（u = 弧长 / 截面周长或带宽，配合 RepeatWrapping 可重复贴图）；'stretch' = 一张贴图从头铺到尾（u 归一化到 [0,1]）
tube 模式专用
radiusnumber0.1管道半径（≤ 0 时该条跳过）
radialSegmentsnumber8圆周分段数（越大越圆滑，最小 2）
startRadnumber0截面起始角偏移（弧度）
generateStartCapbooleanfalse是否生成起点封盖（需 radialSegments ≥ 3）
generateEndCapbooleanfalse是否生成终点封盖（需 radialSegments ≥ 3）
plane 模式专用
widthnumber0.1带宽（≤ 0 时该条跳过）
side'both' | 'left' | 'right''both'带相对中心线的偏侧：'both' 居中、'left' / 'right' 仅一侧
sharpbooleanfalse是否在锐角拐角做几何修补（避免带面撕裂）
arrowbooleanfalse末端是否生成箭头

关于 UV：管道的 v 沿圆周 [0,1]，u 沿路径方向；扁平带的 v 沿带宽 [0,1]，u 沿路径方向。uvMode 只影响 u。'repeat' 模式下若要贴图重复，需把材质贴图的 wrapS 设为 THREE.RepeatWrapping。

Methods

.dispose(): void

释放所有路径几何体；若材质由本组件创建则一并释放（外部传入的 material 不会被释放）。

示例

import { Path } from '@cyc/3d-components/core';

// 3D 折线 → 圆管（拐角倒圆角 + 起终点封盖）
const tube = new Path({
  paths: [{
    path: [[0,0,0],[3,0,0],[3,3,0],[0,3,0]],
    mode: 'tube', bevelRadius: 0.5, radius: 0.15,
    generateStartCap: true, generateEndCap: true,
  }],
});
scene.add(tube);

// 2D 折线 → 扁平带（锐角修补 + 末端箭头）
const route = new Path({
  paths: [{
    path: [[0,0,0],[3,0,0],[3,0,3],[0,0,3]],
    mode: 'plane', bevelRadius: 0, width: 0.5,
    side: 'both', sharp: true, arrow: true, up: [0,1,0],
  }],
});
scene.add(route);

// 闭合环形管道（首尾自动相连，标架同步）
const ring = new Path({
  paths: [{
    path: [[2,0,0],[0,0,2],[-2,0,0],[0,0,-2]],
    mode: 'tube', close: true, radius: 0.1, radialSegments: 12,
  }],
});
scene.add(ring);
