快速开始

安装

本库依赖 three 和 gsap（peer dependencies）:

npm install three gsap
npm install @cyc/3d-components

引入方式

支持整包引入和按需引入（tree-shaking），兼容 ESM / CJS：

整包 (ESM)

import { BaseGroup, HeatMesh, ShinyMaterial, Util } from '@cyc/3d-components';

按需引入（推荐）

import { BaseGroup } from '@cyc/3d-components/core';
import { HeatMesh } from '@cyc/3d-components/heat';
import { ShinyMaterial } from '@cyc/3d-components/material';
import { Util } from '@cyc/3d-components/utils';

CommonJS

const { BaseGroup } = require('@cyc/3d-components/core');
const { Util } = require('@cyc/3d-components/utils');

快速示例

import * as THREE from 'three';
import { BaseGroup } from '@cyc/3d-components/core';
import { ShinyMaterial } from '@cyc/3d-components/material';

const widget = new BaseGroup({ name: 'widget', scale: 1.5 });
widget.add(new THREE.Mesh(
  new THREE.SphereGeometry(0.5, 32, 32),
  new ShinyMaterial({ color: 0x3366ff, metalness: 0.3 })
));
scene.add(widget);

BaseGroup

extends THREE.Group implements IUpdatable, IDisposable

通用 Group 包装器。接受声明式配置，自动处理子节点添加、缩放、资源释放。适合作为所有 group 类组件的基础。

Constructor

new BaseGroup(options?: BaseGroupOptions)

Parameters — BaseGroupOptions

参数类型默认值说明
namestring—Object3D.name
scalenumber1初始缩放 (uniform)
visiblebooleantrue是否可见
userDataobject—自定义数据
childrenObject3D[]—子节点数组，自动 add()

Methods

.update(delta: number): void

每帧调用。在子类中覆盖实现逐帧行为。默认空操作。

参数类型说明
deltanumber距上一帧秒数（上限 100ms）
.dispose(): void

释放所有资源。遍历子节点释放 geometry / material，清空 children。覆盖时最后调用 super.dispose()。

Wall

extends THREE.Group implements IDisposable

墙体绘制组件。接收一组墙体数据，为每面墙生成带圆角拐角、可选墙洞的挤出几何体。路径定义在 XZ 平面（y 被忽略），墙体从 y = 0 向上生长。适合建筑户型、园区围栏、室内隔墙等场景。

Constructor

new Wall(options: WallOptions)

Parameters — WallOptions

参数类型默认值说明
wallsWallData[]—必填。一组墙体数据，每个元素绘制一面墙
materialTHREE.Material—共享材质（所有墙体复用）。不传则使用默认 MeshStandardMaterial
namestring—Object3D.name
visiblebooleantrue是否可见
userDataobject—自定义数据
childrenObject3D[]—额外子节点，自动 add()

Data — WallData（单面墙）

字段类型默认值说明
path[number, number, number][]—墙体中心线路径，XZ 平面坐标 [x, y, z]（y 被忽略）。路径定义走向，墙体按 width 向两侧加厚
widthnumber—墙体厚度（向路径两侧各加厚 width / 2）
heightnumber—墙体高度（沿 Y 轴挤出）
radiusnumber | number[]0拐角圆角半径。数值 ⇒ 所有拐角统一；数组 ⇒ 按 path 顶点索引逐拐角单独指定（元素 0 = 直角）。小于 width/2 的半径自动按直角处理
radiusSegmentsnumber8圆角分段数（越大越圆滑）
closebooleanfalse首尾是否闭合（闭合时形成环形墙体）
holeWallHole[]—墙洞（窗 / 门）列表，画在墙的立面上，按 segment 归属到对应墙段并贯通墙体厚度

Data — WallHole（单个墙洞 · 窗 / 门）

墙洞画在墙的立面上（沿墙方向 × 高度），沿墙体厚度方向贯通。每个墙洞归属于某一段墙体，用该段的局部 2D 坐标描述轮廓。

字段类型默认值说明
segmentnumber—所属墙段索引（段 i = path[i] → path[i+1]）
path[number, number][]—轮廓，该段局部坐标 [x, y]：x = 沿直墙段距离（从段起点计），y = 高度（0 为墙底）。首尾自动闭合
radiusnumber0拐角圆角半径
radiusSegmentsnumber8圆角分段数

Methods

.dispose(): void

释放所有墙体几何体；若材质由本组件创建则一并释放（外部传入的 material 不会被释放）。

示例

import { Wall } from '@cyc/3d-components/core';

const wall = new Wall({
  walls: [
    {
      // 闭合矩形房间，带圆角
      path: [[0, 0, 0], [6, 0, 0], [6, 0, 5], [0, 0, 5]],
      width: 0.25,
      height: 3,
      radius: 1,
      radiusSegments: 16,
      close: true,
      hole: [
        // 第 0 段（[0,0,0]→[6,0,0]）上的一扇门：沿墙 2~4m、高 0~2.1m
        { segment: 0, path: [[2, 0], [4, 0], [4, 2.1], [2, 2.1]] },
        // 第 1 段（[6,0,0]→[6,0,5]）上的一扇窗：沿墙 1.5~3.5m、离地 1~1.8m
        { segment: 1, path: [[1.5, 1], [3.5, 1], [3.5, 1.8], [1.5, 1.8]], radius: 0.15 },
      ],
    },
    {
      // 开放隔墙，圆角拐角（首尾不闭合）
      path: [[-2, 0, 8], [2, 0, 8], [2, 0, 11]],
      width: 0.2,
      height: 2.5,
      radius: 0.8,
    },
  ],
});
scene.add(wall);

HeatMesh

extends THREE.Mesh implements IUpdatable, IDisposable

热力图可视化 Mesh。以半透明球体表示热力区域，支持动态强度控制、GSAP 动画、每帧自动脉冲效果。

Constructor

new HeatMesh(options?: HeatMeshOptions)

Parameters — HeatMeshOptions

参数类型默认值说明
namestring—Object3D.name
countnumber64球体顶点分辨率
radiusnumber1热力区域半径
intensitynumber0.5初始强度 0–1
visiblebooleantrue是否可见
userDataobject—自定义数据

Properties

属性类型说明
heatIntensitynumber当前热力强度 0–1
heatRadiusnumber当前热力半径

Methods

.update(delta: number): void

每帧更新，应用基于 heatIntensity 的脉冲透明度效果。

.setIntensity(value: number): this

设置热力强度。自动钳制到 0–1。返回 this 支持链式调用。

参数类型说明
valuenumber新强度值，越界自动钳制
.dispose(): void

释放 geometry 和 material。

ShinyMaterial

extends THREE.MeshStandardMaterial

预配置的高光 PBR 材质。基于 MeshStandardMaterial，提供合理的金属/粗糙度默认值。所有修改方法支持链式调用。适合产品展示、UI 光泽元素。

Constructor

new ShinyMaterial(options?: ShinyMaterialOptions)

Parameters — ShinyMaterialOptions

参数类型默认值说明
colorColorRepresentation0xffffff基础颜色 (hex / CSS / THREE.Color)
metalnessnumber0.1金属度 0 (塑料) → 1 (金属)
roughnessnumber0.2粗糙度 0 (镜面) → 1 (磨砂)
emissiveColorRepresentation0x000000自发光颜色
emissiveIntensitynumber0自发光强度

Methods

.setColor(color: ColorRepresentation): this

设置基础颜色。接受 hex、CSS 字符串或 THREE.Color 实例。自动 needsUpdate。

.setShininess(metalness: number, roughness: number): this

同时调整金属度和粗糙度。自动 needsUpdate。

参数类型说明
metalnessnumber金属度 0–1
roughnessnumber粗糙度 0–1

Util

Math / Color / Geometry 三大类工具函数。通过 Util.xxx() 调用，也支持按名单独导入（tree-shaking）。

import { Util } from '@cyc/3d-components/utils';
Util.clamp(5, 0, 10);
Util.createSphere(2, 500);
